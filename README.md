# Modelação Matemática e Cálculo com Python

## 🎯 Sobre o Projeto
Este repositório reúne uma coleção de scripts e *notebooks* em Python focados na modelação matemática e na resolução de problemas avançados de cálculo. Os algoritmos aqui presentes foram desenvolvidos durante o meu percurso em Engenharia Eletrotécnica no Instituto Politécnico de Bragança (IPB).

O foco principal destes códigos é demonstrar a transição da teoria matemática para a aplicação computacional, utilizando métodos numéricos e analíticos para simular sistemas, resolver equações complexas e gerar visualizações precisas de dados aplicados à engenharia.
